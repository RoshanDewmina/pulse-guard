== Description ==

**Saturn Watchdog** monitors your WordPress wp-cron events and alerts you when tasks fail, run late, or behave abnormally.

= Key Features =

* Automatic Event Discovery
* Anomaly Detection
* Health Scoring (A-F grades)
* Multi-Channel Alerts (Email, Slack, Discord)
* Bulk Management for agencies
* Zero Performance Impact

= Perfect For =

* WordPress agencies managing 10-1000+ sites
* Mission-critical WordPress applications  
* E-commerce sites
* Membership sites
* Sites using DISABLE_WP_CRON

== Installation ==

1. Go to Plugins → Add New
2. Search for "Saturn Watchdog"
3. Click "Install Now" and "Activate"
4. Go to Settings → Saturn Watchdog
5. Enter your organization token
6. Click "Test Connection"

== Frequently Asked Questions ==

= Do I need a Saturn account? =
Yes, create a free account at saturn.example.com.

= Does it work with DISABLE_WP_CRON? =
Yes! Works with both WordPress pseudo-cron and real system cron.

= Will this slow down my site? =
No. Minimal overhead (<10ms).

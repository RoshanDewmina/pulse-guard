# CLI Reference

Complete reference for the `pulse` CLI.

## Commands

### saturn login

Authenticate with Saturn.

\`\`\`bash
saturn login                  # Device flow
saturn login --token YOUR_TOKEN  # Direct token
\`\`\`

### saturn run

Wrap and monitor a command.

\`\`\`bash
saturn run --monitor ID -- COMMAND
saturn run --monitor ID --capture-output -- COMMAND
saturn run --monitor ID --timeout 3600 -- COMMAND
\`\`\`

### pulse monitors

List and manage monitors.

\`\`\`bash
pulse monitors list
pulse monitors create --name "Backup" --interval 3600
pulse monitors get YOUR_MONITOR_ID
pulse monitors delete YOUR_MONITOR_ID
\`\`\`

### pulse incidents

View incidents.

\`\`\`bash
pulse incidents list
pulse incidents get YOUR_INCIDENT_ID
pulse incidents ack YOUR_INCIDENT_ID
pulse incidents resolve YOUR_INCIDENT_ID
\`\`\`

:::info Authentication
All API requests require authentication via Bearer token:

\`\`\`bash
Authorization: Bearer YOUR_TOKEN
\`\`\`

Get tokens from **Settings → Tokens** in the Saturn dashboard.
:::

## Rate Limits

- **Ping API**: 60 requests/minute per monitor
- **Other endpoints**: 1000 requests/hour per organization

## Common Patterns

### Send Success Ping

\`\`\`bash
curl -X POST https://api.saturnmonitor.com/api/ping/YOUR_MONITOR_ID/success \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '\{"durationMs": 12456\}'
\`\`\`

### Send Fail Ping

\`\`\`bash
curl -X POST https://api.saturnmonitor.com/api/ping/YOUR_MONITOR_ID/fail \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '\{"exitCode": 1, "output": "Error message"\}'
\`\`\`

### List Incidents

\`\`\`bash
curl -X GET https://api.saturnmonitor.com/api/incidents?status=OPEN \
  -H "Authorization: Bearer YOUR_TOKEN"
\`\`\`

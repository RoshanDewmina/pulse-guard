# Welcome to Saturn Docs

Saturn is a monitoring platform for cron jobs, scheduled tasks, and background jobs with statistical anomaly detection.

## Quick Links

- [Get Started](/get-started/overview) - Begin monitoring in minutes
- [Kubernetes Integration](/kubernetes/sidecar) - Deploy Saturn sidecar
- [WordPress Plugin](/wordpress/install) - Monitor wp-cron jobs
- [CLI Tools](/cli/install) - Command-line monitoring
- [API Reference](/api/reference) - REST API documentation

## Key Features

- **Anomaly Detection**: Statistical analysis catches performance degradation before failure
- **Health Scores**: A-F grades with MTTR/MTBF tracking
- **Multi-Channel Alerts**: Email, Slack, Discord, and webhooks
- **Zero-Code Integrations**: Kubernetes Helm chart and WordPress plugin
- **Output Capture**: Automatic log capture and redaction

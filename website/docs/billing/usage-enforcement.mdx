# Usage Enforcement

How Saturn enforces plan limits.

## Monitor Limits

- **At 90%**: Warning email sent
- **At 100%**: Cannot create new monitors
- **Existing monitors**: Continue working

## User Limits

- **At limit**: Cannot invite new users
- **Existing users**: Retain access

## Feature Flags

- **Webhooks**: Disabled if not in plan
- **SSO**: Enterprise only
- **Custom redaction**: Enterprise only

## Upgrading

Upgrade anytime via **Settings → Billing**. Changes apply immediately.
